## Начало
### 1) История создания.
Во первых, хочу рассказать, как я пришел к созданию этого репозитория
Все началось с обычного мема из тиктока, который я увидел и решил проверить. Там был прикол про казино, где с помощью библиотеки random выбиралось случайное число от 1 до 100, и если выпадало число №67, то команда os.remove() удаляла папку System32. Конечно, это не правда, сейчас объясню почему. os.remove не может удалять папки, а только файлы. Также, папку System32 удалить обычным скриптом почти невозможно и существуют более легкие способы навредить компьютеру. Один из таких способов - мой скрипт, который очищает разметку дисков и уничтожает мозги windows через реестр с помощью прав администратора
